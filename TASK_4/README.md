# DESCRIPTION

This Python script implements a spam email classifier using Natural Language Processing (NLP) and machine learning techniques. The program begins by importing essential libraries such as pandas and numpy for data handling, NLTK for text preprocessing, and scikit‑learn for model building and evaluation. The dataset spam.csv is loaded, containing email messages labeled as either “ham” (not spam) or “spam.” After cleaning unnecessary columns, the labels are mapped to numeric values: 0 for ham and 1 for spam.
A preprocessing function is defined to normalize the text. It converts all characters to lowercase, removes punctuation, and eliminates common stopwords using NLTK’s stopword list. This ensures that only meaningful words remain for analysis. The cleaned messages are then vectorized using TF‑IDF (Term Frequency–Inverse Document Frequency), which transforms text into numerical features that reflect word importance across the dataset.
The data is split into training and testing sets, with 80% used for training and 20% for evaluation. A Multinomial Naive Bayes classifier, well‑suited for text classification, is trained on the vectorized training data. Predictions are made on the test set, and performance metrics such as accuracy, precision, recall, and a detailed classification report are displayed. These metrics provide insight into how well the model distinguishes spam from ham.
Finally, the script includes an interactive section where users can input their own email text. The input is preprocessed, vectorized, and passed through the trained model to predict whether the email is spam or not. The loop continues until the user types “exit.”
Overall, this project demonstrates the complete pipeline of building a spam detection system: data preprocessing, feature extraction, model training, evaluation, and user interaction. It highlights practical applications of NLP and machine learning in cybersecurity and email filtering, making it a strong internship‑level deliverable.
